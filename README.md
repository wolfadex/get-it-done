# Get It Done

A very basic template for an [Elm](https://elm-lang.org/) project, using [Nix](https://nixos.org/) and [direnv](https://direnv.net/).

## Development

#### Prerequisites

- [Nix](https://nixos.org/)
- [direnv](https://direnv.net/)

#### Setup

- Run `direnv allow`
- Run `npm run dev`
- Open browser to http://localhost:8080
